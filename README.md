# CoreInventory - Inventory Management System

**A modern, full-featured Flask + MySQL inventory management web application** with user authentication, multi-warehouse support, receipts, deliveries, stock adjustments, and a real-time dashboard.

Built for small-to-medium businesses and educational projects.

## ✨ Features

- ✅ **User Authentication** – Signup, Login, Forgot/Reset Password (bcrypt)
- ✅ **Product Management** – CRUD + category, SKU, unit, stock alerts
- ✅ **Receipts (Incoming)** – Create, view, auto-update stock + ledger
- ✅ **Deliveries (Outgoing)** – Create, view, auto-decrease stock + ledger
- ✅ **Stock Adjustments** – Add/deduct/correct stock with reason
- ✅ **Internal Transfers** – Between warehouses
- ✅ **Dashboard** – Total products, current stock value, low-stock alerts
- ✅ **Stock Ledger** – Full movement history
- ✅ **Multi-Warehouse Support**
- ✅ **Responsive Tailwind UI** (modern cards, tables, modals)

---

## 🛠 Tech Stack

- **Backend**: Flask, SQLAlchemy, PyMySQL
- **Database**: MySQL
- **Auth**: bcrypt + flask-cors
- **Frontend**: Pure HTML + Tailwind CSS + Vanilla JS (fetch API)
- **Other**: python-dotenv, Flask-Mail (ready for future), particles.js
